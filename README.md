# drone-object-detection
``` YOLOv8 fine-tuned on VisDrone for aerial drone object detection. No manual dataset download needed — handled automatically. Detects pedestrians, cars, buses, trucks, tricycles and more. Simple Streamlit app to upload drone footage and get annotated results instantly. ```
--run train.py first to train the model 
--then run app.py using streamlit run app.py command to run the file
